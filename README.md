## Endpoints

| Endpoint | کارکردن |
|---|---|
| `GET /` | زانیاری API |
| `GET /api/info?url=` | زانیاری پۆست یان ڕیلس |
| `GET /api/video?url=` | لینکی ڤیدیۆ/ڕیلس |
| `GET /api/audio?url=` | لینکی ئۆدیۆ |
| `GET /api/profile?user=` | زانیاری پرۆفایڵ |

---

## نموونەی بەکارهێنان

```
GET https://jackinsta.vercel.app/api/video?url=https://www.instagram.com/reel/xxx
GET https://jackinsta.vercel.app/api/profile?user=cristiano
```
